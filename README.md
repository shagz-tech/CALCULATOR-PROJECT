# CALCULATOR-PROJECT
🧮 NeoCalc – Scientific Calculator
NeoCalc is a stylish, responsive, and modern scientific calculator with light/dark mode toggle, sound effects, animation, and keyboard support. Built using HTML, CSS, and JavaScript – no frameworks, just pure web tech.

🚀 Features
✅ Scientific functions (sin, cos, tan, log, √, π, e, ^)

✅ Dark/Light theme with smooth switch animation

✅ Click sound on button press

✅ Button press animation

✅ Keyboard input support

✅ Scrollable history of calculations

✅ Responsive design for mobile/tablets
